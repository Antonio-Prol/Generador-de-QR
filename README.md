#  App generador de QR

Aplicación en Python para generar códigos QR de forma rápida y sencilla. Ideal para principiantes que quieren aprender sobre interfaces gráficas y manipulación de imágenes.

## Tecnologías utilizadas
- Python 3
- Tkinter
- qrcode
- Pillow (PIL)

## Instalación
```bash
git clone https://github.com/Antonio-Prol/Generador-de-QR.git
cd qr-generator-app
pip install -r requirements.txt
python qr_generator.py
